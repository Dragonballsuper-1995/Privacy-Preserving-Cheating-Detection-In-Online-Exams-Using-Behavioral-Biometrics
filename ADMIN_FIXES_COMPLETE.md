## 🎯 What Was Fixed

### 1. ✅ **Risk Scoring Formula - Now Documented**

**Formula Display:** Hover over "ℹ️ Formula" in the Sessions panel header

**The Formula:**
```
Final Risk Score = (Behavioral × 0.35) + (Anomaly × 0.35) + (Similarity × 0.30)

Where:
Behavioral = Average of (Typing, Hesitation, Paste, Focus) scores
Anomaly = Isolation Forest anomaly detection score
Similarity = Answer similarity score (if available)
```

**Example Calculation:**
```
Your Session:
- Typing: 20%
- Hesitation: 65%
- Paste: 85%
- Focus: 65%

Step 1: Calculate Behavioral Score
behavioral = (0.20 + 0.65 + 0.85 + 0.65) / 4 = 0.5875 (59%)

Step 2: Weighted Fusion (simplified, no anomaly/similarity trained yet)
final_risk ≈ behavioral = 59% → Displayed as 60%
```

**Flagging Logic:**
```python
is_flagged = final_risk_score >= 0.50  # 50% threshold

Your case:
60% >= 50% → ✅ FLAGGED
```

### 3. ✅ **Session Categorization - Real vs Simulated**

**New Feature:** Filter buttons in admin dashboard

**Three Options:**
1. **All** - Show all sessions (default)
2. **Real** - Only sessions from actual exams you took
3. **Simulated** - Only auto-generated test sessions

**How It Works:**
- Backend checks if session ID starts with `sim-` or `test-`
- Backend checks for `is_simulated` flag in metadata
- Frontend filters sessions based on selection

**Visual Indicator:** 
- Filter buttons at top right of sessions panel
- Click to switch between views
